# ThreeSumClosest.java

Given an integer array `nums` and an integer `target`, return the sum of the three integers in `nums` such that the sum is closest to `target`.

## Example

**Input:** nums = [-1, 2, 1, -4], target = 1  
**Output:** 2

**Explanation:**
The sum that is closest to the target is 2. (-1 + 2 + 1 = 2)

## Test Cases

- nums = [1, 2, 4, 8, 16, 32, 64, 128], target = 82 → Output: 82
- nums = [-1, 2, 1, -4], target = 1 → Output: 2
- nums = [0, 0, 0], target = 1 → Output: 0

## Complexity
- Time: O(n^2)
- Space: O(1)
