# Equilibrium Index

This directory contains implementations of the Equilibrium Index algorithm in Java, Python, and JavaScript.

## Problem Statement
Given an array of integers, the equilibrium index is an index where the sum of elements at lower indexes is equal to the sum of elements at higher indexes.

### Example
Input: [1, 3, 5, 2, 2]
Output: 2
Explanation: 1 + 3 = 4, 2 + 2 = 4

## Files
- `EquilibriumIndex.java`: Java implementation
- `equilibrium_index.py`: Python implementation
- `equilibriumIndex.js`: JavaScript implementation

## How to Run
- Java: Compile and run the test class using `javac` and `java`.
- Python: Run the test file using `python equilibrium_index_test.py`.
- JavaScript: Run the test file using `node equilibriumIndex.test.js`.
