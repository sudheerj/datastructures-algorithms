# Median of Two Sorted Arrays

## Problem
Given two sorted arrays nums1 and nums2 of size m and n respectively, return the median of the two sorted arrays.

## Test Cases

### Test Case 1: Even total length
- Input: nums1 = [1, 3], nums2 = [2, 4]
- Output: 2

### Test Case 2: Odd total length
- Input: nums1 = [1, 2], nums2 = [3, 4, 5]
- Output: 3

### Test Case 3: One array empty
- Input: nums1 = [], nums2 = [1]
- Output: 1

### Test Case 4: Both arrays empty
- Input: nums1 = [], nums2 = []
- Output: Exception or error

### Test Case 5: Arrays with negative numbers
- Input: nums1 = [-5, 3, 6], nums2 = [-2, 4, 10]
- Output: 3
